# Hakoniwa Chart Catalog

[![View Site](https://img.shields.io/badge/View_Site-Hakoniwa_Chart_Catalog-0f766e?style=for-the-badge)](https://Taniory.github.io/HakoniwaChartCatalog/)

**Hakoniwa Chart Catalog** は、LLM によって生成されたユニークなチャートやデータビジュアライゼーションを収集したギャラリーサイトです。

## 🌟 プロジェクト概要

本サイトは、多様な形式のチャート（棒グラフ、散布図から高度なヒートマップまで）を閲覧できるショーケースです。裏側では LLM がプロンプトに基づき、ECharts を用いた描画スクリプトとダミーデータを全自動で提示しています。

*   **新しい発見**: 異なるテーマや視点から、データをどう可視化するかのアイデアが追加されます。
*   **セキュアな描画環境**: LLM が生成した JavaScript は、アプリケーション本体から分離された Sandbox (`iframe` + `CSP`) 内でのみ安全に実行・描画されます。
*   **ソース展開**: 気になるチャートの描画スクリプトや実際のJSONデータ構造を、サイト上で直接確認できます。

## ⚠️ 生成AIの利用について

本プロジェクトに掲載されているコンテンツ（データセットの内容、解説テキスト、描画用コード）は、生成 AI によって作成されています。
*   表示されているデータやトレンドは架空のダミーデータです。
*   生成結果は安全検証を通していますが、ブラウザ環境によっては意図しないレイアウト崩れが発生する可能性があります。

## 💻 開発環境とローカル実行

フロントエンドは React + Vite で構成されており、生成された静的ファイルは GitHub Pages にホスティングされています。ローカルでの閲覧・開発を行う場合は以下の手順を実行してください。

```bash
# クローンと依存パッケージのインストール
npm ci

# 開発用サーバーの起動
npm run dev
```
